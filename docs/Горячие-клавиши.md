# Горячие клавиши

## Глобальные команды (работают в любой вкладке)

Эти сочетания настраиваются в `chrome://extensions/shortcuts`:

| Сочетание | Действие |
|-----------|----------|
| **MediaPlayPause** | Воспроизведение / Пауза |
| **MediaTrackPrevious** | Предыдущий трек |
| **MediaTrackNext** | Следующий трек |
| **MediaStop** | Стоп |
| **Alt+Z** | Пипетка (eyedropper) — выбрать цвет с экрана |

> **Примечание:** Для работы медиа-клавиш VK должен быть открыт хотя бы в одной вкладке.

## Клавиши на странице VK

| Сочетание | Действие |
|-----------|----------|
| **Ctrl+Q** | Переключение раскладки (en ↔ ru) для поля ввода |
| **Esc** | Закрыть панель настроек расширения |

## Внутри панели настроек

| Действие | Описание |
|----------|----------|
| **Стрелки ←↑↓→** | Перемещение тени блока / позиции часов (в соответствующих редакторах) |
| **Ctrl+Z / Ctrl+Y** | Undo / Redo (история изменений темы) |

## Настройка глобальных клавиш

1. Откройте `chrome://extensions/shortcuts` (или `edge://extensions/shortcuts`)
2. Найдите **VK Styles Clean**
3. Назначьте удобные сочетания для каждой команды
4. Для медиа-клавиш выберите «Глобальный» охват

> **Совет:** Если медиа-клавиши на клавиатуре перехватываются другим приложением (Spotify, Windows Media), проверьте, что VK Styles имеет приоритет в настройках расширений.
