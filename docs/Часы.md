# Часы

Виджет часов, который отображается на странице VK.

## Включение

Нажмите переключатель **«Часы»** в панели настроек.

## Настройки

| Параметр | Описание |
|----------|----------|
| **Цвет** | Цвет текста часов (через пикер) |
| **Размер часов** | Ползунок размера цифр |
| **Размер даты** | Ползунок размера строки даты |
| **Прозрачность** | Ползунок прозрачности виджета |
| **Скорость разделителя** | Скорость мигания `:` между часами и минутами |

## Формат

| Опция | Варианты |
|-------|----------|
| **Дата** | Без даты / Короткая / Полная |
| **Секунды** | Показывать / Скрыть |
| **Формат** | 12-часовой / 24-часовой |

## Визуальные эффекты

Доступно несколько **эффектов отображения** — переключение через соответствующую кнопку.

## Шрифт

По умолчанию используется системный шрифт. Можно задать **собственный шрифт** (название или URL на Google Fonts).

## Расположение

| Функция | Описание |
|---------|----------|
| **Пришпилить к фону** | Часы «приклеиваются» к фону и не перекрывают интерфейс |
| **Перетаскивание** | Зажмите и перетащите часы в любое место на странице |
| **Позиция (X/Y)** | Точная подстройка положения стрелками или drag-and-drop |
