# Фон и обои

## Установка фона

1. Откройте панель настроек → раздел **«Обои»** (кнопка с иконкой картинки)
2. Вставьте ссылку на изображение или видео в поле ввода и нажмите **Enter**
3. Или выберите из **истории ранее использованных фонов** (сетка миниатюр внизу)

## Поддерживаемые форматы

### Изображения
- Прямые ссылки на изображения (jpg, png, webp, gif)
- Документы VK (загруженные в «Документы»)

### Видео
- **YouTube** — ссылки вида `youtube.com/watch?v=...` или `youtu.be/...`
- **Vimeo** — ссылки вида `vimeo.com/...`
- **VK Video** — ссылки на видео ВКонтакте
- **Прямые ссылки** на mp4, webm, m3u8

## Эффекты фона

В разделе обоев доступны ползунки эффектов:

| Эффект | Описание |
|--------|----------|
| **Размытие блоков** | Размытие фона за контентными блоками (эффект «стекла») |
| **Контрастность** | Контрастность фонового изображения |
| **Насыщенность** | Насыщенность цветов фона |
| **Яркость** | Яркость фонового изображения |
| **Размытие фона** | Общее размытие фоновой картинки |
| **Поворот тона** | Сдвиг цветового тона (отрицательное значение = авто-вращение) |
| **Инверсия** | Инверсия цветов фона |
| **Сепия** | Эффект старой фотографии |
| **Параллакс** | Движение фона при движении мыши (отрицательное = инверсия) |
| **Виньетка** | Затемнение по краям экрана |
| **Шум** | Зернистость поверх фона |

## Слайдшоу

Если в истории обоев есть несколько изображений, можно включить **автоматическую смену**:

| Интервал | Описание |
|----------|----------|
| 1 сек – 1 час | Фиксированные интервалы |
| При смене трека | Фон меняется при переключении аудиозаписи |

Также доступен режим **случайного порядка**.

## Размер и позиция фона

### Размер
- **Заполнить** (Fill) — фон заполняет весь экран
- **По экрану** (Screen) — вписывается в экран
- **Обои** (Wallpaper) — реальный размер
- **Растянуть** (Stretch) — растягивается по экрану
- **Замостить** (Tile) — повторяется плиткой

### Позиция
9 вариантов: центр, углы, края (верх, низ, лево, право).

## Отражение фона

- Без отражения
- Горизонтальное
- Оба направления
- Вертикальное

## Звук видео-фона

Для видео-обоев доступно управление громкостью (ползунок) и кнопка mute/unmute.

## Режим «Только фон»

Кнопка скрывает весь контент страницы, оставляя только фоновое изображение/видео — для наслаждения обоями в полноэкранном режиме.
