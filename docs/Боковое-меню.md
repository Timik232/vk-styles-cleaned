# Боковое меню

## Фон меню

**Переключатель «Фон меню»** добавляет полупрозрачный фон к боковой панели навигации с эффектом размытия (backdrop-filter) и скруглёнными углами.

Работает как со старым (`#side_bar`), так и с новым (`#layout_sidebar`) макетом VK.

## Мини-режим

**Переключатель «Мини меню»** сворачивает боковое меню до **иконок** — текстовые подписи, счётчики, разделители и юридический текст внизу скрываются.

При наведении курсора на иконку — появляется **тултип** с названием пункта.

Ширина мини-меню: **44px** (иконки 36×36px).

## Закреплённое меню

**Переключатель «Закрепить меню»** делает боковое меню **sticky** — оно остаётся на месте при прокрутке страницы, всегда доступно.

## Цвет меню

Настройка цвета текста и иконок в боковом меню:

| Режим | Описание |
|-------|----------|
| **Контрастный** | Автоматический контрастный цвет |
| **Из акцента** | Берёт основной акцентный цвет |
| **Свой цвет** | Ручной выбор через HSL-ползунки |
| **Из текста** | Использует общий цвет текста страницы |
