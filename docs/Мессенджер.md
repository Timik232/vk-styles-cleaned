# Мессенджер

Настройки для улучшения мессенджера VK.

## Функции

### Список чатов справа

Перемещает **список диалогов** из левой колонки в правую — ближе к тексту переписки.

- Переключатель: **«Чат-список справа»**
- CSS-класс: `html.rm`

### Открытие из уведомлений

Входящие сообщения открываются **в самом мессенджере**, а не во всплывающих мини-окнах.

- Переключатель: **«Мессенджер из уведомлений»**

### Нижние иконки действий

Иконки действий в переписке (прикрепить фото, видео и т.д.) перемещаются ниже, чтобы не занимать место у поля ввода.

- Переключатель: **«Нижние иконки»**

### Быстрые реакции

Быстрый доступ к **эмодзи-реакциям** на сообщения — наведение на сообщение сразу показывает кнопку реакции.

- Переключатель: **«Быстрые реакции»**

### Подсветка непрочитанных

Выделяет **непрочитанные диалоги** заметным цветом фона.

- Переключатель: **«Подсветка непрочитанных»**
- Цвет подсветки: **настраиваемый** через цветовой пикер

| Параметр | Описание |
|----------|----------|
| Включение | Переключатель «Подсветка непрочитанных» |
| Цвет | Любой цвет через встроенный пикер |
| По умолчанию | Полупрозрачный акцентный цвет |

### Скрытие диалогов

Через контекстное меню мессенджера можно **скрыть отдельные диалоги** из списка. Скрытые ID хранятся в настройках и не отображаются в списке чатов.
