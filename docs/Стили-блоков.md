# Стили блоков

## Варианты стилей

Доступно **5 вариантов** оформления контентных блоков:

| Стиль | Описание |
|-------|----------|
| **Стандартный 1** | Обычные блоки с небольшим скруглением (4px) |
| **Стандартный 2** | Аналогично, альтернативный вариант |
| **Скруглённый 1** | Сильно скруглённые углы (20px), плавные формы |
| **Скруглённый 2** | Аналогично, альтернативный вариант |
| **Без отступов** | Блоки без промежутков, занимают всю ширину, 0 скругление |

Переключение: нажмите на **«Стиль блоков»** в панели настроек для перебора вариантов.

## Тень блоков

Добавляет **тень** к контентным блокам. Настройки:

| Параметр | Описание |
|----------|----------|
| **Режим** | Выкл / Из границы / Свой цвет |
| **Тип** | Внешняя (Offset) / Внутренняя (Inset) |
| **Размытие** | Степень размытия тени (ползунок) |
| **Распространение** | Размер тени (ползунок) |
| **Позиция** | Смещение тени по X и Y (перетаскивание точки на контроле) |

## Смещение страницы

Ползунок **«Смещение»** сдвигает весь контент страницы влево или вправо (0–100%). Удобно если нужно освободить часть экрана.
