# Changelog

All notable changes to VK Styles Clean will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [2.0.8.4] - 2026-04-05

### Fixed
- Синхронизирован фикс внешнего вида боковой панели: убран квадратный контейнер при обрезке обводки

### Changed
- Обновлена версия расширения до `2.0.8.4`

## [2.0.8.3] - 2026-04-05

### Fixed
- Синхронизирована версия релиза после фикса стробов в основной (Chromium) ветке

### Changed
- Обновлена версия расширения до `2.0.8.3`

## [2.0.8.2] - 2026-04-05

### Fixed
- Исправлена обводка активного пункта в боковой панели: возвращено ограничение контейнера `#layout_sidebar.stLayoutWrapper__sidebarFixed`, чтобы свечение не выходило за границы

### Changed
- Обновлена версия расширения до `2.0.8.2`

## [2.0.8.1] - 2026-04-05

### Fixed
- Исправлены стробы фона: возвращена рабочая формула яркости, чтобы фон снова светился ярче в такт музыке

### Changed
- Обновлена версия расширения до `2.0.8.1`

## [2.0.8] - 2026-04-05

### Added
- Добавлен фильтр фотоальбомов с быстрым переключением между всеми альбомами, обоями и темами
- Добавлен браузер live wallpaper с выбором альбомов и видео в панели настроек

### Changed
- Обновлена версия расширения до `2.0.8`
- Обновлены метаданные Firefox-пакета и связанные документы под сборку `2.0.8`

### Fixed
- Обновлены селекторы аудиостраницы для новых вариантов плеера VK
- Добавлены новые селекторы блоков рекомендаций и промо для обновленной верстки VK
- Улучшена загрузка видеоальбомов и поведение браузера live wallpaper
- Исправлена функция `Lg()`: теперь она выбирает самый новый подходящий аудиоключ по `_vNNN`
- Исправлен false positive в GitHub Actions для разрешенного использования `atob()` внутри `atou()`

## [2.0.7.1] - 2026-03-19

### Fixed
- Migrated Firefox startup and command handling to Promise-based `browser.*` APIs
- Removed unsupported `commands.*.global` manifest properties from the Firefox build
- Updated Firefox compatibility metadata and docs to match the current build requirements
- Fixed the visualizer `Strobes` effect so it flashes instead of making the whole screen darker

### Known issues
- Firefox build is still unstable and may not work correctly yet

## [2.0.7] - 2026-03-17

### Fixed
- Исправлена синхронизация линии ползунка громкости VK при изменении громкости колесом/клавишами: UI обновляется вместе со звуком
- Исправлен цвет текста элемента «Новая история» в левом меню: текст больше не становится чёрным и читается в тёмной теме
- Доработан режим «Фикс»: левое меню больше не обрезается слева и не получает внутренний скролл, сохраняя обычный внешний вид

## [2.0.6] - 2025-02-22

### Added
- Ссылка на GitHub-репозиторий в боковой панели расширения (вместо паблика атакующего)
- Сообщение «Поставьте ⭐ звёздочку на GitHub» в нижней части панели и README
- Ссылка на DonationAlerts в README

### Fixed
- Настройка «Текст шапки» теперь корректно меняет цвет текста и кнопок верхнего плеера в новом интерфейсе VK (селектор `data-testid=TopAudioPlayer` вместо устаревшего `#web_spa_top_audio_player`)
- Укорочен текст «Поставьте ⭐ на GitHub» в нижней части панели — текст обрезался
- Наложение элементов меню боковой панели в режиме «Фикс» при скролле (футер перекрывал пункты меню)
- Ошибка "Cannot access contents of the page" в Яндекс Браузере при запуске `executeScript`
- Сочетание Alt+B для переключения фона — добавлен preventDefault/stopPropagation
- Колёсико мыши больше не меняет громкость на всей странице музыки — только на панели плеера
- Регулировка громкости колёсиком при активном эквалайзере/визуализаторе

### Changed
- Удалена кнопка обновления версии из боковой панели
- Удалены элементы «Темы для ВК» и «Живые обои» (ссылались на группу атакующего)
- Удалена видео-галерея из группы атакующего (~70 строк кода)

## [2.0.5] - 2026-02-21

### Fixed
- Исправлена регулировка громкости колёсиком мыши при активном эквалайзере/визуализаторе
- Ползунок громкости VK теперь показывает реальное значение (не застревает на 100%) при активном эквалайзере
- GainNode переведён в passthrough-режим (усиление=1), громкость управляется нативно через VK
- Колёсико мыши теперь регулирует громкость при наведении на всю панель музыки сверху, а не только на ползунок

## [2.0.4] - 2026-02-21

### Fixed
- Исправлено изменение размера эмодзи — обновлены CSS-селекторы для нового DOM VK (`.Emoji`)

### Security
- Сужен `web_accessible_resources` с `["*"]` до минимально необходимых файлов (`2.js`, `css`, `mcss`, `i/*`)
- Удалён мёртвый код `plus()`/`less()` — остатки малварного механизма промо/пейволла оригинального расширения

## [2.0.3] - 2026-02-20

### Fixed
- Фон (изображение и видео) больше не пропадает при переключении вкладок и простое
- Исправлен краш обработчика ошибок VK-видео при отсутствии резервных URL
- VK-видео и документы автоматически перезагружаются при истечении ссылки
- YouTube/Vimeo-фоны перезагружаются при длительном простое вкладки (>30 сек)

## [2.0.2.1] - 2026-02-20

### Fixed

## [2.0.2] - 2026-02-20

### Fixed
- Исправлена работа режима «Мини» — теперь корректно скрывает текст, оставляя только иконки с подсказками
- Исправлен фон боковой панели — больше не обрезает элементы меню
- Исправлена совместимость фона и мини-режима при одновременном включении
- Исправлена битая ссылка на исследование Koi Security в README

### Added

## [2.0.1] - 2026-02-15

### Fixed

- Исправлен фон боковой панели (overflow:hidden)

### Added
- Начальные CSS-правила для режима мини-сайдбара
- GitHub Actions CI (security.yml) и CD (release.yml)
- Contributing guide, VK disclaimer, README

## [2.0.0] - 2026-02-10

### Security
- Удалён удалённый код-загрузчик (`eval`/`Function` через VK-профиль)
- Удалена принудительная подписка на группу злоумышленника
- Удалён сброс настроек каждые 30 дней
- Удалена манипуляция CSRF-токенами (`remixsec_redir`)
- Удалена Яндекс.Метрика с динамическим ID
- Удалён redirect при деинсталляции
- Удалены все обращения к внешним серверам

### Changed
- Ребрендинг в «VK Styles Clean»
- Исправлены эмодзи-реакции и настройки IMU по умолчанию
